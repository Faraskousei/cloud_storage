# 🎯 Demo Guide - Cloud Storage UI/UX

## 🚀 Cara Menjalankan Demo

### 1. **Jalankan Aplikasi**
```bash
# Windows - Cara Cepat
double-click install_and_run.bat

# Atau manual
python run.py
```

### 2. **Akses Aplikasi**
- Buka browser: http://localhost:5000
- Interface akan dimuat dengan animasi yang smooth
